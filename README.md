# Laravel To-Do App

A simple To-Do application built with Laravel for task management.

## Features
- Add, edit, delete, and view tasks.
- Mark tasks as completed.
- Responsive design with modern UI.

## Requirements
- PHP >= 8.1
- Composer
- MySQL
- Node.js & NPM
